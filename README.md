# מרכז הריפואים של יניב

ריפו זה משמש כאינדקס עברי מרכזי לריפוזיטוריז של יניב ב־GitHub.

## קישור חיצוני לאתר

https://yanivmizrachiy.github.io/repo/

## קישור לריפו

https://github.com/yanivmizrachiy/repo

## מה כבר קיים

- אתר HTML בעברית ו־RTL.
- קובץ נתונים `data/index.json` עם 27 ריפואים ציבוריים.
- סימון של 11 ריפואים פרטיים שמוסתרים מהאתר הציבורי.
- הסבר קצר בעברית לכל ריפו ציבורי.
- קטגוריה לכל ריפו.
- סטטוס בסיסי לכל ריפו.
- אחוז מוכנות בסיסי.
- כפתור פתיחה לכל ריפו בגיטהאב.
- חיפוש בעברית.
- מיון לפי שם, ביקור אחרון, עדכון ואחוז מוכנות.

## מקור הנתונים

האתר טוען את הנתונים מתוך:

```text
data/index.json
```

## פרטיות

ריפואים פרטיים אינם מוצגים באתר הציבורי. האתר מציג רק את מספר הריפואים הפרטיים המוסתרים.

## חשוב

הנתונים הנוכחיים הם Bootstrap עברי ראשוני. בשלב הבא צריך לשפר את GitHub Action כך שיסרוק אוטומטית כל ריפו לפי README, RULES, STATE, workflows וקבצי מקור, ואז יעדכן את ההסברים בצורה מדויקת יותר.
