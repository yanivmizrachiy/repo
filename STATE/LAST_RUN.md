- צריך לשדרג את GitHub Action כך שיסרוק אוטומטית README, RULES, PROJECT_RULES, STATE, workflows וקבצי מקור.
- צריך להוסיף דוחות Markdown נוספים כמו `REPORTS/ALL_REPOS_SUMMARY.md`.
- צריך לוודא GitHub Pages פעיל בהגדרות הריפו ומציג את האתר מהענף `main`.
